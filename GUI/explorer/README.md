# Explorer

Enterprise-grade GUI for Explorer. Provides a reference front-end for inspecting network state and interacting with Synnergy services.

## Setup

```bash
make install
```

## Development

- `make build` – compile TypeScript sources
- `make test` – run unit tests
- `make lint` – lint codebase
- `make format` – format code
- `make docker-build` – build production container

## Environment

Configuration is driven by `.env` variables. See `.env.example` for defaults.
